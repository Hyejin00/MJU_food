# MJU_food
명지대 주변 맛집을 보여주고 간단한 랜덤 뽑기 기능이 있는 앱. 2-1 학기 기말 프로젝트 제출물.

점심 메뉴가 고민되는 명지대 학우들을 위해 앱을 통해 주변 맛집 정보를 제공한다. 또한 여럿이서 하나의 점심 메뉴를 정할 때 의견이 모아지지 않는 경우, 랜덤 뽑기의 기능도 제공하고 있다.

## 사용 기술
Java
